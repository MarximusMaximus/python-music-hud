# python-music-hud-tv

<https://github.com/MarximusMaximus/pytest-music-hud-tv>

by Marximus Maximus (<https://marximus.com>)

A TV HUD to display song information for an dance event.

## Features

- Event title card.
- Event title card with current and next track titles.
- Display current, next, and following tracks with dance style types.
- Blanking out surprise track names.
- Hiding a track's info altogether.
- Displaying special messages in place of next and following, based on current
    track

## Usage

1. Install `python-music-hud-tv` via your favorite method, e.g.:
    - `pip install python-music-hud-tv`
    - `poetry add python-music-hud-tv`

### Additional Info

Configure constants to desired behavior.
